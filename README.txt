Alternate Reality Remake (Modern Demo v0.5)
-------------------------------------------
Python 3 + Pygame demo inspired by the 1980s classic.

Controls:
  W/S - Move forward/back
  A/D - Turn left/right
  M   - Toggle map
  I   - Inventory
  T   - Toggle day/night
  1-4 - Choose menu option
  ESC - Exit / leave establishment
